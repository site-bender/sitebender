# sitebender

[![GitHub license](https://img.shields.io/badge/license-MIT-06f.svg)](https://github.com/site-bender/sitebender/blob/main/LICENSE.txt) [![npm version](https://img.shields.io/npm/v/@sitebender/sitebender?style=flat&color=3c3)](https://www.npmjs.com/package/@sitebender/sitebender) [![PRs Welcome](https://img.shields.io/badge/PRs-welcome-c6c.svg)](https://sitebender.org/) [![Build and Test](https://github.com/site-bender/sitebender/actions/workflows/build_and_test.yml/badge.svg?color=3c3)](https://github.com/site-bender/sitebender/actions/workflows/build_and_test.yml) ![Coveralls](https://img.shields.io/coverallsCoverage/github/site-bender/sitebender)


Entire web apps from a JSON configuration object, including:

- calculations
- conditional display
- validation
- injection of values from many possible sources (asynchronously)
- all HTML elements
- common components
- injection and de-duping of scripts/stylesheets
- and more coming, including a no-code site builder.

Note: **This is beta code. Documentation, etc. is lacking. Coming soon.**
