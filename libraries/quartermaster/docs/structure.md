## Rules

1. ALL FILES ARE `index.ts` OR `index.tsx`
2. NAMES GO ON FOLDERS
3. PascalCase for components/modules
4. DEFAULT EXPORTS for functions/components
5. NAMED EXPORTS only for types and constants
6. NO BARREL FILES
